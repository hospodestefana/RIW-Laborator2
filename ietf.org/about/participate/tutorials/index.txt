Tutorials are organized by the IETF Education, Mentoring and Outreach Directorate, and generally conducted by current IETF participants. Tutorial topics include the processes and tools used by the IETF community to create Internet standards, updates on technical topics of current interest to IETF participants, and an overview of work underway in the IETF Areas. A complete list of tutorials can also be found on the IETF Datatracker. Process oriented tutorials Process oriented tutorials include how to bring new work into the IETF, tips on how to use tools to create documents such as Internet-Drafts and RFCs, and how to more effectively participate in IETF meetings. IETF Area overview tutorials IETF Area overview tutorials provide regular updates from the various IETF Areas on recent developments, as well as work that is currently underway. Technical tutorials Technical tutorials cover a wide range of topics of interest to IETF participants and relevant to work underway within the IETF. Newcomers' tutorials These tutorials provides introductions to a number of aspects of the Internet Engineering Task Force (IETF) history, structure and operations.
