In this section Hide in this section IETF Area overview tutorials Applications and Real-Time Area introduction Internet Area introduction Operations and Management Area introduction Routing Area introduction Security Area introduction Transport Area introduction Key Info The purpose of the tutorial is not to explain the routing protocols, but to describe the history and purpose of the Routing Area as well as to provide an overview of the current work in the area. It will show how the various working groups relate to each other and to working groups in other areas. Finally, the session will consider how routing-related work in other areas can access and benefit from the expertise available in the Routing Area.
