IETF Area overview tutorials

IETF Area overview tutorials provide regular updates from the various IETF Areas on recent developments, as well as work that is currently underway.

    Presented by IETF Area Directors, these overview tutorials are a great way to get up to speed on work that is underway in each of the IETF Areas.

    • Applications and Real-Time Area introduction

      This tutorial provides a high level overview of the main work areas that the Applications and Real-Time (ART) Area covers.

    • Internet Area introduction

      This tutorial provides a high level overview of the Internet Area, including the core protocol building blocks developed and maintained by the area.

    • Operations and Management Area introduction

      This presentation provides a high-level overview of the Operations and Management Area (OPS).

    • Routing Area introduction

      This tutorial provides an overview of the work and activities in the Routing Area (RTG) of the IETF.

    • Security Area introduction

      This tutorial provides a high-level overview of the Security Area (SEC), including building blocks developed and maintained by the area.

    • Transport Area introduction

      This tutorial provides a high level overview of the Transport and Services (TSV) Area, which works on mechanisms related to end-to-end data transport.
